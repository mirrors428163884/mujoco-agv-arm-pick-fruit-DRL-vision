# 软件著作权说明书

## 一、软件基本信息

### 1.1 软件名称

**基于深度强化学习的农业采摘机器人控制软件**（英文名称：Picking_Sorting）

### 1.2 软件版本

V1.0

### 1.3 开发完成日期

2025年1月

### 1.4 软件分类

应用软件 / 人工智能 / 机器人控制

### 1.5 软件用途

本软件用于训练和控制农业采摘机器人，实现在复杂农业场景中的自主导航、目标果实定位和精确采摘操作。软件采用深度强化学习技术，训练移动操作机器人（包括移动底盘、6自由度机械臂和16自由度灵巧手）完成从接近目标到精确抓取的完整采摘任务。

## 二、软件功能说明

### 2.1 主要功能概述

本软件实现了基于两阶段深度强化学习的农业采摘机器人智能控制系统，主要功能包括：

1. **两阶段训练框架**：将复杂的采摘任务分解为"接近"和"抓取"两个独立训练阶段
2. **智能路径规划**：训练机器人自主规划最优接近路径
3. **精确抓取控制**：训练灵巧手实现多指协同的稳定抓取
4. **视觉感知处理**：处理深度图像信息用于环境感知和决策
5. **域随机化训练**：通过多种噪声模拟提高模型泛化能力
6. **课程学习机制**：渐进式难度提升的智能训练策略

### 2.2 详细功能描述

#### 2.2.1 Stage 1（接近阶段）训练功能

本阶段训练机器人的移动底盘和机械臂协同工作，实现从任意初始位置接近目标果实并到达预抓取姿态。

**主要功能点：**

- **底盘控制**：控制双阿克曼转向底盘的线速度，实现平稳移动
- **机械臂控制**：输出6个关节的角度增量，控制末端执行器位置和姿态
- **目标追踪**：根据视觉和状态信息追踪目标果实位置
- **预抓取姿态判定**：判断是否到达适合抓取的位置和姿态
- **AVP奖励计算**：使用预训练的Stage 2 Critic评估当前位置的"可抓取性"

**输入输出规格：**

| 类型 | 维度 | 组成 |
|------|------|------|
| 状态观测 | 15-16 | 底盘速度(2) + 末端位置(3) + 末端四元数(4) + 末端速度(3) + 目标位置(3) + [可选: 位置有效标志(1)] |
| 深度图像 | 84×84 | 单通道深度图，归一化至0-255 |
| 动作输出 | 8 | 底盘速度(2) + 机械臂关节增量(6) |

#### 2.2.2 Stage 2（抓取阶段）训练功能

本阶段训练机器人的灵巧手完成精确抓取任务，在近距离完成对目标果实的稳定抓握。

**主要功能点：**

- **机械臂微调**：在近距离进行末端位置的精细调整
- **灵巧手控制**：控制16关节灵巧手（12个可控自由度）的抓取动作
- **触觉反馈处理**：处理4个手指（拇指、食指、中指、无名指）的触觉传感数据
- **抓取质量评估**：评估多指接触、力分布、抓取稳定性
- **扰动抵抗测试**：在外力干扰下保持稳定抓握

**输入输出规格：**

| 类型 | 维度 | 组成 |
|------|------|------|
| 状态观测 | 35 | 末端位置(3) + 末端四元数(4) + 末端速度(3) + 机械臂关节(6) + 目标位置(3) + 手部关节(12) + 触觉(4) |
| 深度图像 | 84×84 | 单通道深度图 |
| 动作输出 | 18/20 | [底盘(2, 锁定为0)] + 机械臂关节增量(6) + 手部关节增量(12) |

#### 2.2.3 视觉感知功能

软件集成了完整的视觉处理模块，用于环境感知和决策支持。

**主要功能点：**

- **深度图获取**：从腕部深度相机获取84×84分辨率深度图像
- **深度噪声模拟**：模拟真实传感器的Cutout噪声、椒盐噪声、边缘模糊等
- **视觉有效性检测**：检测深度图有效像素比例，过滤低质量输入
- **CNN特征提取**：使用卷积神经网络提取深度图视觉特征

#### 2.2.4 域随机化功能

软件实现了全面的域随机化机制，提高训练模型的泛化能力。

**随机化类型：**

1. **深度噪声随机化**
   - Cutout遮挡：模拟深度图空洞
   - 椒盐噪声：模拟镜面反射
   - 像素丢失：模拟传感器故障
   - 深度相关高斯噪声

2. **光照随机化**
   - 环境光强度随机
   - 漫反射强度随机
   - 光源方向扰动

3. **物体随机化**
   - 物体形状：球体、圆柱、胶囊、椭球、方块
   - 物体尺寸：可配置的尺寸范围
   - 物体质量：35-75克范围

4. **检测帧丢失模拟**
   - 单帧随机丢失
   - 连续帧丢失
   - 有效性标志输出

#### 2.2.5 课程学习功能

软件实现了渐进式的课程学习机制，逐步提高训练难度。

**Stage 1 课程：**
- 距离初始化从近到远
- 碰撞惩罚从轻到重
- 朝向要求从松到严

**Stage 2 三阶段课程：**
- Phase 0（0-2M步）：基础抓取学习，弱域随机化，近距离初始化
- Phase 1（2M-8M步）：稳定抓取学习，滑移惩罚渐增
- Phase 2（8M步以后）：扰动抵抗学习，随机外力测试

#### 2.2.6 训练监控与日志功能

软件提供完整的训练监控和日志记录功能。

**监控功能：**

- **WandB集成**：在线训练曲线监控和实验对比
- **TensorBoard支持**：本地训练日志可视化
- **检查点保存**：自动保存最优模型权重
- **可视化验证**：支持MuJoCo实时渲染查看训练效果

## 三、软件技术特点

### 3.1 核心技术创新

#### 3.1.1 AVP（非对称价值传播）技术

本软件创新性地提出了AVP（Asymmetric Value Propagation）技术，使用预训练的Stage 2 Critic为Stage 1提供"可抓取性"奖励信号。

**技术特点：**

1. **势能整形奖励**：采用`r_avp = λ × (γ·Φ(s') - Φ(s))`公式，奖励"向可抓取状态移动"而非"处于可抓取状态"
2. **OOD检测**：实现视觉有效性检测和MC Dropout不确定性估计，避免误导性奖励
3. **自适应Lambda调度**：从预热期到衰减期的四阶段权重调度策略

#### 3.1.2 Progress-Based奖励设计

重构的奖励系统采用"Progress + Terminal + Regularization"三阶段结构：

1. **进度奖励**：基于距离变化而非绝对距离，消除"站桩刷分"行为
2. **终止奖励**：成功/失败的大额奖惩，明确任务目标
3. **正则化惩罚**：存活惩罚和停滞惩罚，促进积极探索

#### 3.1.3 分离输出头网络架构

Actor网络采用分离输出头设计：

- **Stage 1**：底盘输出头（2D）和手臂输出头（6D）独立
- **Stage 2**：手臂输出头（6D，σ=-2.5更稳定）和手部输出头（12D，σ=-1.5更多探索）

#### 3.1.4 可选GRU记忆模块

Stage 1网络支持可选的GRU记忆层，用于处理物体位置观测的帧丢失问题，使网络能够利用历史信息推断当前状态。

### 3.2 算法特点

#### 3.2.1 PPO强化学习算法

软件采用PPO（Proximal Policy Optimization）算法进行策略优化，具有以下特点：

- **稳定训练**：通过裁剪目标函数限制策略更新幅度
- **高效采样**：支持多环境并行采样加速训练
- **值函数归一化**：使用RunningMeanStd进行输入归一化

#### 3.2.2 关节空间控制

采用关节空间直接控制方式：

- 直接输出关节角度增量（Δθ）
- 避免逆运动学（IK）不稳定性
- 动作范围限制确保安全运行

### 3.3 仿真环境特点

#### 3.3.1 MuJoCo物理引擎

基于MuJoCo 3.2.3+物理引擎构建高精度仿真环境：

- 精确的接触动力学模拟
- 快速的物理计算性能
- 支持EGL无头渲染

#### 3.3.2 机器人模型

完整建模的移动操作机器人系统：

- **移动底盘**：Ranger Mini V2双阿克曼转向底盘
- **机械臂**：xArm6 6自由度机械臂
- **灵巧手**：LEAP Hand 16关节灵巧手
- **传感器**：腕部深度相机 + 4指触觉传感器

## 六、软件主要技术

### 6.1 深度强化学习技术

#### 6.1.1 PPO算法实现

软件实现了完整的PPO（Proximal Policy Optimization）算法：

**核心组件：**

1. **Actor-Critic网络**
   - Actor：输出动作分布的均值和标准差
   - Critic：估计状态价值函数

2. **GAE优势估计**
   - 使用广义优势估计计算优势函数
   - 参数：gamma=0.99, tau=0.95

3. **裁剪目标函数**
   - 限制策略更新幅度，保证训练稳定性
   - 裁剪参数：epsilon=0.2

4. **经验缓冲区**
   - 存储采样的状态、动作、奖励等数据
   - 支持多环境并行采样

#### 6.1.2 奖励函数设计

**Stage 1 奖励函数：**

| 类别 | 奖励分量 | 公式 | 权重 |
|------|----------|------|------|
| Progress | EE进度奖励 | k_ee × (d_prev - d_curr), clip到±0.2 | 3.0 |
| Progress | 底盘进度奖励 | k_base × (err_prev - err_curr), clip到±0.2 | 2.0 |
| Regularization | 存活惩罚 | -0.01 per step | -0.01 |
| Regularization | 停滞惩罚 | -0.1 if stagnating N steps | -0.1 |
| Conditional | 朝向奖励 | k_ori × max(0, cos(θ)), 仅d_ee<0.3m时激活 | 0.4 |
| Terminal | 预抓取成功 | +50 | 50.0 |
| Terminal | 碰撞惩罚 | -50 | -50.0 |
| Terminal | 超时惩罚 | -20 | -20.0 |
| Optional | AVP奖励 | λ × (γ·Φ(s') - Φ(s)), clip到±0.2 | 自适应 |

**Stage 2 奖励函数：**

| 类别 | 奖励分量 | 公式 | 权重 |
|------|----------|------|------|
| Progress | EE进度奖励 | k × (d_prev - d_curr), clip到±0.2 | 3.0 |
| Progress | 里程碑奖励 | +2.0 if d_ee < 0.05m | 2.0 |
| Grasp Quality | 多指计数 | k_cnt × (n_contact / 4) | 1.5 |
| Grasp Quality | 力范围奖励 | 力在[f_low, f_high]范围内得分 | 1.0 |
| Grasp Quality | 力平衡奖励 | -k_bal × variance(forces) | 0.5 |
| Grasp Quality | 手指协同 | 拇指+其他对握奖励 | 1.0 |
| Penalty | 滑移惩罚 | -k_s × |v_rel| | 课程自适应 |
| Penalty | 冲击惩罚 | -penalty if v_ee > 0.3 at first contact | -5.0 |
| Terminal | 成功奖励 | +100 | 100.0 |
| Terminal | 失败惩罚 | -50 | -50.0 |
| Perturbation | 扰动测试 | +10 / -5 | ±10.0 |

### 6.2 神经网络架构

#### 6.2.1 Stage 1 网络结构

```
输入层:
├── 状态输入 (15-16维)
└── 深度图输入 (1×84×84)
         │
    ┌────┴────┐
    │ CNNBase │
    │ Conv(1→32,k3)+ReLU+Pool │
    │ Conv(32→64,k3)+ReLU+Pool │
    │ Conv(64→64,k3)+ReLU+Pool │
    │ AdaptiveAvgPool(4×4) │
    │ Flatten→Linear→256+ReLU │
    └────┬────┘
         │
    Concatenate (状态 + CNN特征)
         │
    [可选: GRU层 (hidden=128)]
         │
    ┌────┴────┐
    │         │
 Actor MLP  Critic MLP
 [256, 128] [256, 128]
    │         │
┌───┴───┐    │
│       │    │
base(2) arm(6)  Value(1)
σ_base  σ_arm
```

#### 6.2.2 Stage 2 网络结构

```
输入: [State(35) | Depth(7056)] = 7091维

├── Split ──┬────────────────┐
│           │                │
```

### 6.3 物理仿真技术

#### 6.3.1 MuJoCo仿真引擎

软件基于MuJoCo物理引擎构建仿真环境：

**技术特点：**

1. **高精度接触模拟**：精确计算多体接触力学
2. **高效计算性能**：支持GPU加速的并行仿真
3. **EGL无头渲染**：支持服务器端无显示器渲染
4. **MJCF模型格式**：标准化的机器人模型描述

#### 6.3.2 机器人运动学与动力学

**底盘控制**：
- 双阿克曼转向模型
- 速度控制范围：[-4, 4] m/s

**机械臂控制**：
- 6自由度关节控制
- 关节增量范围：[-0.05, 0.05] rad/step
- PID位置控制器

**灵巧手控制**：
- 16关节（12可控自由度）
- 关节增量范围：[-0.05, 0.05] rad/step
- 四指触觉反馈

### 6.4 计算机视觉技术

#### 6.4.1 深度图像处理

**图像获取**：
- 腕部深度相机模拟
- 84×84分辨率
- 单通道深度值归一化至0-255

**噪声模拟**：
- Cutout遮挡：模拟深度空洞
- 椒盐噪声：模拟镜面反射
- 像素丢失：模拟传感器故障
- 边缘模糊：模拟深度边界不确定性

#### 6.4.2 CNN特征提取

**Stage 1 CNN：**
- 3层卷积 + 自适应池化
- 输出256维特征向量

**Stage 2 CNN：**
- 3层卷积（8×4步长、4×2步长、3×1步长）
- 自适应池化 + 全连接
- 输出256维特征向量
